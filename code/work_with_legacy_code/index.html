Group Methods(写出所有方法名和访问权限) - 2. Look at Hidden Methods - 3. Look for Deciions that can change. - 4. Look for internal relationships（写出所有变量和方法使用的变量，画出调用关系，根据把调用关系分成的几个大区域来拆分class）. - 5.
