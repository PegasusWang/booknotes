# 第1章：投资的快与慢

- 通货膨胀
- 合理的预期
- 复利的价值(复利；投资要趁早；稳健投资的价值和优势)
- 容易被忽略的投资常识
  - 企业利润高增长等于高收益？买入业绩告诉增长的资产还是亏了钱。股票市场收益：企业的钱；情绪的钱
  - 投资收益无法线性实现。投资者必须战胜通货膨胀和不确定性的波动性
- 如何定义成功的投资
  - 长期业绩是否跑赢市场基准。没跑赢指数不妨考虑指数基金
  - 是否解决了实际的问题
  - 实际收益率为正。扣除通货膨胀
  - 时间和精力投入
- 致富的耐心。棉花糖实验。不愿意慢慢富裕的人大多数成为资本市场的牺牲品。你到底多有耐心


# 第2章：资产配置中最重要的事情

- 集中投资之殇。即便正确了 1000 次，只要失败一次满盘皆输。活着最重要，其次才是盈利。
- 资产组合的分散化。防守才是最好的进攻，避免出现大幅回撤。分散式最投资者最好的保护。股票组合；定投分散时间
- 资产配置思路。马科维茨资产组合理论。分散不相关的资产之间，可以增加收益，减少风险
- 风险与回报比。凯利公式
- 经济周期简析。繁荣期-衰退期-萧条期-复苏期
- 周期晴雨表：主要经济指标。PMI指数；市场利率；长期国债利率；广义货币M2；通胀(PPI/CPI)
- 市场估值的定义与应用。选择长期资产首要工作，应该始于对估值的判断
  - 绝对估值法（贴现法）
  - 相对估值法PE/PB/PS/PEG
    - 市盈率PE: 公司当前总市值除以全年总净利润
    - 市净率PB: 每股股价与每股净资产的比率
    - 股息率。每股总派息金额与每股股价的比率
    - 净资产收益率ROE。公司净利润与净资产的比率
  - 估值工具的应用和误区。通过简单务实的量化筛选增加投资胜率
  - 估值指标在指数上的应用。全市场指标可以帮助投资者衡量当前市场的整体估值水平和市场热度
  - 估值百分位的局限性。均值回归
    - 时间跨度。行业历史，多时段对比
    - 多指标结合。
    - 估值失真。可能出现水分
    - 估值回归的有效性。
- 动态再平衡的核心逻辑。
  - 格雷厄姆在聪明的投资者提到50/50股债配置，当偏离的时候进行动态再平衡，长期的配置比例维持50/50。适量卖出超涨品种，买入超跌品种逆向操作
  - 均值回归理论。均值回归定律，是再平衡的核心逻辑。减少回撤，降低波动性，心理压力也小很多
  - 再平衡建立在长期不会死的品种之上。比如指数组合
  - 普通投资者先从基础的股票和债券开始。之后扩展到股票、债券、黄金、现金。
  - 动态再平衡不复杂，但是反人性的逆向操作，需要持之以恒。很难长期执行下去，最终领导这份犒赏的人少之又少


# 第3章：资产配置工具
- 如何定义财富。
  - 有形的财富：资本。相对安全：房产、政府债券、存单、黄金、现金。市场风险类：股票、投资性房产、商品、实业。没有绝对安全
  - 无形的财富。人力资本
- 股票权益类。股票。长期打败大部分资产，但是波动大。多元化配置对冲风险
- 债券固收类。和股票负相关性，提供对冲作用
  - 市场利率和通胀风险。债券的价格和市场利率走势相反。个人投资者不建议配置长期债券
  - 发行人偿债能力。政府和企业
  - 流动性因素
  - 汇率风险
- 投资性房产和REITs
- 大宗商品
- 贵金属。控制在 20%甚至 10%以内。实物金条；黄金基金；衍生类投资
- 现金资产。3-6 个月生活开销。方式：银行存款；国债逆回购；货币基金；套利交易
- 化繁为简。指数化配置。优势：分散化投资；成本低廉；风格不漂移；可繁可简
- 中国资产篇。沪深 300+中证 500
- 海外资产。优先美、德、日这种全球经济占比高的国家。财富到达一定规模；计划移居海外；国内资产全面泡沫化的时候。


# 第4章：生命周期下的资产配置计划
- 了解自己：不一样的人生。
  - 你的职业: 收入中间高、两头低。职业是否青春饭
  - 你的性格：自律的投资方式会逐步改善投资的性格，并把我们塑造成更卓越的投资者
    - 了解投资的常识。多理解历史和投资常识，在投资时会趋于理性
    - 丰富投资阅历。先前投资经历潜移默化影响未来的投资情绪。入市就顺风顺水对于塑造成熟的投资性格并非好事
    - 保持开放的心态
- 如何决策资产配置框架
  - 设定可行的投资目标
  - 制定长期战略框架。资金要与期限匹配。考虑流动性
  - 中短期战术配置。
  - 定期回顾和总结。充分了解投资收益或者亏损是来源于市场，还是决策失误，或者意外事件
- 设定长期的投资目标
  - 准备什么时候退休
  - 养老金的替代率
  - 退休储备金的提取率
  - 退休时我们需要多少钱：未来的家庭开支水平、通货膨胀水平、投资收益率
- 个人投资者的仓位管理
  - 盘点可投资的资金。权益类资产（风险资产）；债券或者现金类（安全资产) 。仓位统计
  - 设置仓位区间。
    - 目标生命周期策略：100 法则，100减去你的年龄是可投资权益资产的比例
    - 仓位浮动区间。动态再平衡；设置区间调整买入和卖出
    - 根据市场估值动态调整
    - 是否有必要融资。投资收益率大于融资成本；未来现金流可以覆盖这笔融资；波动率；贷款和投资的久期；杠杆工具其他用途


# 第5章：实战三部曲
- 年富力强：全力定投
  - 享乐主义还是节俭主义
  - 有效积累财富的方式：定投。持仓上的分散和时间上的分散
  - 定投。长期坚持，持仓成本低于均值；省时省力、高性价比的盈利策略；保持上佳的投资心态；极高的普适性和容错率
  - 定投的钝化。均衡配置或者定投降为辅助策略甚至停止
  - 定投进阶：价值平均策略。将资产市值的逐年增长率设定为远期投资目标，定期不定额
- 人到中间：追求均衡。40-55 岁
  - 如何有效控制回撤。衡量仓位与风险。增加现金类头寸，是降低风险最佳途径。均衡投资
  - 何为均衡投资。
    - 投资三分法：1/3存银行，1/3购置房产，1/3购置股票债券等有价证券。
    - 时间三分法：长期、中期、短期有序结合
    - 家庭资产象限图：4321；40%保本升值；30%钱生钱；20%保命钱；10%要花的钱
    - 50/50股债均衡。在此基础上四等分：股票、债券、现金、黄金等
    - 中年资产配置建议: 稳定且长期贡献更多现金流的资产
- 退休阶段：CPPI 保守型投资组合
  - 保守型投资要点：适量权益资产；低波动性（动态再平衡）；现金流；流动性；提取率
  - CPPI(恒定比例组合): 仓位设定10%-50%区间。风险最小化至少持有 5%-10%股票；保本资产的确定（兼顾更多资产大类）;放大倍数的调整(专业机构的养老产品)
- 各阶段的过度和止盈技巧
  - 定投如何止盈。成功定投三要素：稳定现金流；至少一轮牛熊；合适止盈。
    - 指数水位法。行业估值高低
    - 年化回报率。长期年化收益目标达到后分批止盈
    - 价值平均策略
    - 综合性策略
  - 人生各阶段的资产配置过度


# 第6章：超额收益策略
- 什么是超额收益。beta和阿尔法收益
- 个人投资者的优势
  - 消息面：个人知道的时候为时已晚
  - 技术分析。无法对抗机构
  - 资金面：无定价权
  - 时间和专注力：个人可以专注于长期
- 中国资本市场特点。个人投资者占比高、弱有效性
- 逆向投资策略。
  - 何为逆向投资：人弃我取，人取我与
  - 逆向投资需要解决的问题：
    - 时间：闲钱投资、长期盈利
    - 回撤：未知的震荡和回撤幅度
    - 孤独：模糊的正确胜过精确的错误。
- 超额收益之基金折价溢价。套利
- 超额收益之认购新股
- 构建永续股票投资组合
  - 指数基金还是股票组合。
    - 知识层面：指数投资简单很多
    - 风险与收益比。根据预期和承受力找到一个平衡点
    - 时间和精力。青年人尽量简化投资方式，更多时间留给工作学习和生活。定投或者逢低布局指数是一种高性价比策略
    - 情绪控制
  - 构建攻守俱佳的股票组合
    - 股票数量：8-15 只
    - 市值加权还是等权。建议采纳加权法
    - 调仓管理。出赢保亏，处置效应
    - 选股核心逻辑。适度分散、风格均衡、超额品种、逆向交易、动态平衡
- 多因子量化模型。主流因子包括：价值、成长、红利、波动率、质量等。 smart beta指数产品
- 期货和期权等衍生工具
- 降低你的交易频率。投资组合品种尽量简化，专注于大级别的行情和定期的动态再平衡，使投资更轻松合理和高效
- 设计超额收益策略需注意事项
  - 超额收益策略都有周期性
  - 策略容纳的资金池有限
  - 时间和精力的付出
  - 每年跑赢多少才是合理预期


# 第7章：资产配置的重要细节
- 细分行业的布局建议
  - 商业模式
  - 周期属性
  - 投资行业指数的要点
- 核心和卫星投资理念
  - 何为核心和卫星：核心承担主要风险和收益（稳定）；卫星使用高弹性、高预期的策略为组合贡献超额收益
  - 核心卫星策略应用
- 适度简化：需要配置多少个品种。投资初期指数配置控制在3-8个品种
- 主动被动之争。主动型基金的短板：追涨杀跌；冠军魔咒；运营成本
- 优秀指数品种的 11 个特质：
  - 代表性；相对完整性；指数合理性；成立时间足够长；方便查询数据；可投资性；可替代品和衍生工具；指数的标的数量；流动性充
  裕；低成本；独立性和客观性
- 动态再平衡的方法和频率
  - 定期再平衡。时间周期比如每个季度、每年执行。9-15 个月时间，心理层面更舒服
  - 临界再平衡。震荡超过临界值进行再平衡的调仓干预
- 回测的误区。只关注近期回报率高的产品
- 战胜自己：论情绪管理
  - 独立思考。成功的投资只属于少数人。坚持投资体系，静待市场理性回归，保持自己的节奏不被打乱
  - 自知之明。过度自信导致交易频繁。承受力低的人喜欢豪赌。坦诚承认过自己犯过的错误，是投资必经之路
  - 资本思维。恐惧是思维的杀手，短期波动不是永久性损失。防止自己情绪失控，等待对手情绪失控
  - 量化交易。最好投资 70%心态、20%技术、10%运气。量化你的计划，有助于在投资过程中更好控制情绪做到知行合一。
